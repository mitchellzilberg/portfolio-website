Case Study: myFlix App

Overview

To build a client-side and back-end application using the MERN stack.

Context

This myFlix app was the first project in Careerfoundry's web development course that required both front and back end knowledge of Javascript development.

Objective

To build a complete full-stack project from scratch to add to my professional portfolio and to familiarize myself with the MERN stack.

Duration

October 2020 - December 2020

Tools, Skills, & Methodologies

MongoDB, Express.js, React.js, Node.js

The Process

Back-end

The first part of the process was to create a REST API that interacts with a database and stores data about different movies using MongoDB and Node.js. Users should be able to make requests to the server and receive information about movies, directors, and genres. Users should also be able to register an account, edit their user information, delete their account, and add movies to their personal list of favourites.

Front-End

This part of the development process was dedicated to creating a user-friendly interface that is used for making requests and displaying responses from the server. The client-side of this app consists of multiple views created using the REACT framework.

Looking Back

This project turned out to be one of the most ambitious and challenging aspects of Careerfoundry's web development course. Although the process was difficult and frustrating at times, it also forced me to learn the full scope of the MERN stack. I learned how to use React to create single-page applications as well as creating/interacting with my own database and server.
